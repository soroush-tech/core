## @soroush.tech/bench@2.0.0

Raises the minimum supported Node.js version. No source or API changes — the major bump
reflects a tightened runtime contract. (#266)

### BREAKING CHANGES

- **Node.js engines floor raised `>=18.0.0` → `>=24.18.0`.** Consumers running the
  `soroush-bench` CLI on Node < 24.18.0 must upgrade before installing this release.

### Packaging

- Dev-dependency bumps: `tsdown` `^0.22.4` → `^0.22.7`.
