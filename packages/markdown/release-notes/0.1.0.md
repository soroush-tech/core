## @soroush.tech/markdown@0.1.0

Initial release — the headless markdown editor + renderer, extracted from
`@soroush.tech/design-system` into its own package.

### Features

- Headless, composable markdown parts importable from the barrel: `Control` (state provider),
  `Toolbar` (formatting buttons, heading/code selects, table picker), `Editor` (source textarea
  with Tab-to-indent, selection, and caret restore), and `Preview` (a standalone renderer of
  markdown to design-system primitives, built on `react-markdown`, `remark-gfm`, and
  `rehype-highlight`).
- Per-component theming: the roots register the `MarkdownEditor`, `MarkdownPreview`, and
  `MarkdownToolbar` slots on `@emotion/react`'s `ThemeComponents`, so consumers customize them
  through design-system's `createTheme` / `theme.components`.
- Builds on `@soroush.tech/design-system` (peer) for its primitives, `styled` engine, and theme.
