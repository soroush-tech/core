## @soroush.tech/playwright-coverage@3.0.0

Raises the minimum supported monocart-coverage-reports version. No source or API changes —
the major bump reflects a tightened peer-dependency contract. (#266)

### BREAKING CHANGES

- **`monocart-coverage-reports` peer floor raised `^2` → `^2.12.12`.** Consumers on
  monocart-coverage-reports < 2.12.12 must upgrade to >= 2.12.12 before installing this
  release.

### Packaging

- Dev-dependency bumps: `tsdown` `^0.22.4` → `^0.22.7`.
