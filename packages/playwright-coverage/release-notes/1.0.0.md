## @soroush.tech/playwright-coverage@1.0.0

First release. Collect **per-test V8 JS coverage** from a Playwright e2e run and aggregate it
into a source-mapped **lcov** report via
[monocart-coverage-reports](https://github.com/cenfun/monocart-coverage-reports).

### What's in it

- **Auto fixture** — starts/stops `page.coverage` around every test (Chromium only) and writes
  each test's raw V8 dump to disk.
- **`globalSetup`** — clears stale raw coverage before the run.
- **`globalTeardown`** — merges every raw dump into one lcov report, source-mapped back to your
  original `.ts`/`.tsx`.
- **Glob scoping** — `include`/`exclude` match the repo-relative source path (cwd-stripped,
  forward slashes), so `src/**` globs work cross-platform and Codecov can map paths. Falls back
  to `report.sourceFilter` for full monocart control.
- **Inert when disabled** — gate on an env flag; every piece is a no-op unless `enabled` is
  `true`, and a no-op on non-Chromium browsers, so normal cross-browser e2e runs stay fast and
  uninstrumented.

### Install

```sh
pnpm add -D @soroush.tech/playwright-coverage @playwright/test monocart-coverage-reports
```

`@playwright/test` (`^1`) and `monocart-coverage-reports` (`^2`) are peer dependencies.
