## @soroush.tech/playwright-coverage@2.0.0

Raises the minimum supported Playwright version. No source or API changes — the major bump
reflects a tightened peer-dependency contract. (#259)

### BREAKING CHANGES

- **`@playwright/test` peer floor raised `^1` → `^1.61.1`.** Consumers on Playwright < 1.61.1
  must upgrade to >= 1.61.1 before installing this release.

### Packaging

- Dev-dependency bumps: `@types/node` `^26.1.0` → `^26.1.1`, `@vitest/coverage-v8` `^4.1.9` →
  `^4.1.10`, `eslint` `^10.6.0` → `^10.7.0`, `tsdown` `^0.22.3` → `^0.22.4`, `vitest` `^4.1.9`
  → `^4.1.10`.
