## @soroush.tech/design-system@0.1.0

Initial release — the soroush.tech design system extracted from the web app into a
standalone package.

### Features

- Token-driven `light`/`dark` themes with the full component set: layout & surfaces
  (View, Flex, Grid, Paper, Card, AppBar, Drawer), content & data display (Typography,
  Link, Image, Icon, Table, CodeBlock, Quote, Avatar, Skeleton), inputs & forms
  (Button, ButtonGroup, TextInput, NativeSelect, Select, Checkbox, Radio, Switch,
  ToggleButton, Form, Field, FormControl), feedback (CircularProgress, LinearProgress),
  and overlay primitives (Modal, Popover, Portal, Backdrop, FocusTrap).
- Headless markdown editor & renderer (`Markdown`): Control, Toolbar, Editor, and
  Preview built on `react-markdown`, `remark-gfm`, and `rehype-highlight`.
- Engine barrel re-exporting `@soroush.tech/styled-system` and Emotion primitives
  (`styled`, `css`, `keyframes`, `useTheme`, `Theme`, `createShouldForwardProp`).

### Packaging

- Dual ESM/CJS build via tsdown with per-component subpath exports
  (`@soroush.tech/design-system/Button`, `…/themes`, `…/colors`, …).
- `react`, `react-dom`, `@emotion/react`, and `@emotion/styled` are peer dependencies;
  `@tanstack/react-form` is an optional peer (type-only, used by `Field`).
